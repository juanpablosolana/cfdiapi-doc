¡Bienvenido a cfdi API!

Nuestra api te permite conocer el estatus de las facturas emitidas por el SAT MX haciendo una consulta GET, solo necesitas la siguiente información:

  • ✅ RFC del emisor
  • ✅ RFC del receptor
  • ✅ Folio fiscal
  • ✅ Monto total
